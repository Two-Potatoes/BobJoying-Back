### 🙆‍♀️ 내 손 안에 냉장고 🙆‍♂️

프로젝트 기간 : 2024/01/31 ~

🙋‍♀️ 우리 서비스는요...

1. 냉장고의 **식재료를 관리**해줘요! 
2. 냉장고의 식재료로 할 수 있는 **레시피를 등록하고 관리**할 수 있어요!
3. 냉장고를 공유하는 그룹원에게 먹고 싶은 **레시피를 요청**할 수 있어요!
4. **식사 기록**이 가능하고 그룹원들의 오늘 식사 기록을 **피드**에서 확인할 수 있어요!
